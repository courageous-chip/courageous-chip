## Getting Started

### Installing

```shell
$ yarn install
```

### Starting

```shell
$ yarn start
```

### Compiling

```shell
$ yarn compile
```

### Formatting

```shell
$ yarn format
```

### Linting

```shell
$ yarn lint # --fix
```
